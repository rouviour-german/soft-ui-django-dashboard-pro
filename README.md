# soft-ui-django-dashboard-pro

An innovative, professional admin dashboard boilerplate built with Material-UI, React, and Django. This project provides an "elite" starting point for developers looking to build data-driven applications with a premium look and feel.

## Key Features
- **Premium Design**: Built with Material-UI and Soft UI Design principles.
- **Full-Stack Ready**: Integrated with a robust Django REST API backend.
- **Authentication**: JWT-based login, register, and logout flow.
- **Rich Elements**: Over 70 individual components for rapid UI development.

## Project Architecture
- **API Backend**: Django / DRF / SQLite / Docker.
- **Frontend UI**: React, Redux, Redux-Persist, Material-UI.

## Quick Start
1. **API Server**: `cd django-api && docker-compose up --build`.
2. **React UI**: `cd react-ui && npm install && npm start`.
